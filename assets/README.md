# Assets Directory

Place your app logo here.

## Logo Requirements

- **File name**: `logo.png` (or update `LOGO_PATH` in `config.js`)
- **Recommended size**: 200x200px to 400x400px
- **Format**: PNG with transparent background (preferred)
- **Aspect ratio**: Square (1:1) works best

## Usage

1. Add your logo file to this directory
2. Update `LOGO_PATH` in `config.js` if using a different filename
3. The logo will automatically appear in the landing page header
